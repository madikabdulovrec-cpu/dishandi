СТРУКТУРА ПАПОК
───────────────────────────────────────────────────────────

📁 01_photos/         — Фото парных, интерьеров, фасада
   Подпапки можно создавать сами: например, "парная_1",
   "купель", "зона_отдыха", "снаружи". Или просто кидайте
   всё подряд — я разберу.
   ✓ Чем больше — тем лучше (10–30 фото оптимально)
   ✓ Желательно горизонтальные, минимум 1200px по ширине
   ✓ Форматы: JPG, PNG, WEBP

📁 02_logo/           — Логотип
   ✓ PNG с прозрачным фоном (если есть)
   ✓ Векторный SVG / AI / EPS (если есть)
   ✓ Если логотипа нет — напишите название, нарисую сам

📁 03_info/           — Текстовая информация
   Заполните файл info_template.txt (он уже там лежит).
   Можете заполнять прямо в нём или скинуть в любом виде:
   word, гугл-док, заметка, текст в чате — всё подойдёт.

📁 04_reviews/        — Отзывы
   ✓ Скриншоты с Яндекс.Карт / 2ГИС / Google
   ✓ Или просто текстом в файле reviews.txt
   ✓ 5–10 штук достаточно

📁 05_videos/         — Видео (если есть)
   ✓ Обзор комплекса, ролики из соцсетей
   ✓ Не обязательно, но повышает конверсию
   ✓ Или ссылки на YouTube/Rutube в файле links.txt

📁 06_references/     — Референсы (необязательно)
   ✓ Сайты конкурентов, которые нравятся
   ✓ Скриншоты сайтов, которые вам визуально по душе
   ✓ Просто ссылки в файле links.txt
