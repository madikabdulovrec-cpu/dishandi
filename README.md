# Dishandi SPA — Лендинг

Премиум-лендинг банного комплекса Dishandi SPA (Алматы) для запуска контекстной рекламы.

## Что внутри

- `index.html` — единая HTML-страница со всеми стилями и скриптами
- `banya_content/` — фото, прайс, материалы

## Демо

[Открыть демо →](https://madikabdulovrec-cpu.github.io/dishandi/)

## Стек

- HTML5 / CSS3 (без сборщиков и фреймворков)
- Шрифт: Manrope (Google Fonts)
- Палитра: тёплый эспрессо + золото
- Адаптивная вёрстка от 360px
